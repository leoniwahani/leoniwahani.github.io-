Apa itu Cyber Security?

Cyber security adalah tindakan untuk melindungi sistem, jaringan, dan program dari serangan digital. Serangan ini biasanya ditujukan untuk mengakses, mengubah atau menghancurkan informasi penting, memeras uang pengguna, atau mengganggu proses bisnis perusahaan. Mungkin tantangan terbesar dari cyber security adalah perkembangan teknologi yang terus-menerus. Hal itu berefek pada semakin banyaknya peluang bagi para penjahat cyber untuk melakukannya kejahatan dengan cara-cara baru lainnya. Di sisi lain para ahli cyber security juga akan terus menciptakan solusi untuk menutup potensi bagi penjahat cyber. Dengan demikian keamanan juga akan terus berkembang dan diperbaharui.
